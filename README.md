# LP-EB-TUCODIGO-TuNombre

## Descripción
Landing Page Responsive para **GamerZone Store**, tienda especializada en venta de productos y accesorios para videojuegos. El proyecto centraliza el catálogo, las ofertas, las marcas comercializadas y un formulario de contacto para los distintos segmentos de usuarios (gamers casuales, gamers competitivos/streamers, creadores de contenido, empresas e instituciones educativas).

## Tecnologías utilizadas
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript puro (validación del formulario de contacto)

## Justificación del diseño
- **Paleta de colores:** fondo oscuro (`#0D0D0D` / `#1A1A1D`) con acentos morado (`#7B2FF7`) y cian (`#00F0FF`), transmitiendo tecnología, innovación y alto rendimiento, propio de la identidad gamer.
- **Tipografía:** fuentes modernas y legibles (Rajdhani/Poppins para títulos, Inter/Roboto para texto general).
- **Disposición visual:** estructura de secciones (header, hero, catálogo, ofertas, marcas, creadores/empresas, contacto, footer) basada en las 4 Historias de Usuario y el mockup de Figma, priorizando jerarquía visual y buen contraste para accesibilidad.

## Cómo ejecutar el proyecto localmente
1. Descarga o clona el repositorio.
2. Coloca tus imágenes dentro de `assets/images/` con los nombres indicados en el código (`imagen1.jpg` a `imagen10.jpg`/`.png`).
3. Abre `index.html` directamente en tu navegador (doble clic), o usa la extensión "Live Server" de VS Code para verlo con recarga automática.
